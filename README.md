# Homebrew Tap

Custom Homebrew formulas and casks.

## Installation

```bash
brew tap misbahsy/tap
```

## Available Packages

### cc-bridge (CLI)

Bridge messaging platforms (Telegram, Discord) to Claude Code.

```bash
brew install cc-bridge
```

Then run:
```bash
ccb setup
ccb start
```

### CCB (Desktop App)

Menu bar app for managing cc-bridge.

```bash
brew install --cask ccb
```

## Updating

```bash
brew update
brew upgrade cc-bridge
brew upgrade --cask ccb
```
